MFC 之ComboBox的使用

CComboBox * pComboBox =(CComboBox *)GetDlgItem(ID_ComboBox);
插入数据pComboBox->InsertString(index,_T());
当index 为0时 按照插入顺序，越后插入的index越低
index为-1时，则是插入列表的末尾
pComboBox->AddString 则按照英文字典顺序由上到下排列
VS2005中 将type改为sample 为编辑框,dropdown为可编辑下拉框,dropList为不可编辑下拉框
pComboBox->getWindowText(strText)  获取当前内容到strText中
pComboBox->SetWindowText(strText)  将strText设为当前内容
int index =pComboBox->getCurSel()  返回被选中项的索引
pComboBox->SetCurSel(index) 设置当前内容为索引index的内容

Map用法

map.clear() 使用前 应判断 map.size() 否则会崩溃
map.count(key)  判断key是否有对应的值
map[key] =value 赋值
在insert插入的同时，还有返回值来说明是否插入成功，就是pair< map<string,int>::iterator,bool> >
map.insert(make_pair(key,value)) 
rent= m_map.insert(make_pair("hello",5)）;
rent->second即是成功与否的标志；rent->first就是返回的map<string,int>::iterator迭代器；rent->first.first就是string类型的数据。
