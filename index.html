hexo、gitpages搭建博客

发表于 2017-07-02 | 分类于 编程

gitpages创建

打开github官网
注册github账号
建立Repository  PS:命名规范为
    用户名.github.io
打开所建项目点击Settings
将Github Pages下面的Source选择为master branch 并点击Save

软件安装

GitBash
https://git-scm.com/downloads
NodeJs
https://nodejs.org/en/

配置SSH keys

####检查现有ssh key：####

打开命令行输入:
cd ~/. ssh 检查本机的ssh密钥
如果提示：No such file or directory 说明你是第一次使用git。

####生成新的SSH Key：####

 ssh-keygen -t rsa -C "邮件地址@youremail.com"
效果如下:
Generating public/private rsa key pair.
Enter file in which to save the key (/Users/your_user_directory/.ssh/id_rsa):<回车就好>
之后：
Enter passphrase (empty for no passphrase):<输入加密串>
Enter same passphrase again:<再次输入加密串>
PS：输入的时候在界面上不会显示*,此密码用作提交项目时输入

####添加SSH Key到github：####

打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件
如果看不到可以点开此电脑->查看->勾选隐藏项目
此电脑->查看->选项->查看：显示隐藏文件
取消隐藏受保护的文件
右击Documents and Settings->属性->安全->高级->修改权限->勾选Everyone完全控制

复制私人密钥到github系统中的Account Settings--->SSH Public keys ---> add another public keys key文本框中并点击Add Key

####测试####
命令行中输入ssh -T git@github.com
输入yes以及用户名密码

####设置用户信息
git config –global user.name “cnfeat”//用户名,该名字是提交项目时记录的名字
git config –global user.email “cnfeat@gmail.com”//填写自己的邮箱
PS:必须为英文，如果失败则输入 git init

###Hexo ###

####安装hexo ####
命令行: npm install hexo-cli -g
新建一个文件夹命名 Hexo
在此文件夹中右键打开Git Bash
hexo init
npm install server –save
输入 hexo server -p5000
打开http://localhost:5000 PS:如果打不开可以换一个端口号
打开Hexo目录下的_config.yml
deploy下面配置如下:
type: git
repository: git@github.com:username/创建仓库时输入的名称.github.io.git
branch: master

####部署 ####
hexo clean
hexo g
npm install hexo-deployer –save
hexo d
此时就可以打开github系统 Settings 下面的网址查看成果

###Hexo主题更换####
在Hexo下打开GitBash输入:
git clone https://github.com/iissnan/hexo-theme-next themes/next
将Hexo目录下的_config.yml theme后面改成 next

hexo clean
hexo g
hexo d
此时就更换完成了

打开主页时可能会出现 标签和分类页面错误的情况 此时
在Hexo下打开GitBash输入
hexo new page "tags"
hexo new page "categories"

打开Hexo目录的Source 在categories下的index.md配置如下
---
title: 分类
type: "categories"
comments: false
---
在tags下的index.md配置如下
---
title: 标签
type: "tags"
comments: false
---
